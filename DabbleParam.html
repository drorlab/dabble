DabbleParam.amber module¶

This module contains the AmberWriter class and associated methods, which outputs a prmtop/inpcrd file with CHARMM names and parameters for use with simulation with the AMBER molecular dynamics package. It does this by using the chamber functionality of ParmEd API.

class DabbleParam.amber.AmberWriter(molid, tmp_dir, forcefield='charmm', lipid_sel='lipid', hmr=False, extra_params=None)[source]¶

Bases: object

Creates AMBER-format input files, using either CHARMM or AMBER parameters.

When using the CHARMM parameters, creates a psf/pdb file and interfaces with ParmEd chamber command to create AMBER format files. When using the AMBER parameters, creates a pdb file and runs the amber lipid conversion script to create leap input files.

DabbleParam.charmm module¶

This module contains the CharmmWriter class and associated methods, which outputs a psf/pdb file with CHARMM names and parameters. It does this by converting atom names to CHARMM names, writing intermediate files as necessary to invoke the vmd psfgen plugin.

class DabbleParam.charmm.CharmmWriter(tmp_dir, molid, lipid_sel='lipid', extra_topos=None)[source]¶

Bases: object

An object that handles all the conversions to a psf file by interfacing with psfgen.

Writes a pdb/psf file pair from the current molecule using the CHARMM36 topology and atom names/types. Interfaces with psfgen by dynamically generating the .tcl file that psfgen takes as input. Prompts the user for additional topology files and helps with matching atom names that cannot be automatically translated to the charmm naming conventions.

Attributes:
file (file handle): Temporary file to write TCL script that invokes
psfgen

tmp_dir (string): Directory where temporary files are stored psf_name (str): Prefix for the pdb/psf output files, extension will

be appended

molid (str,optional): the VMD molecule id to write. Defaults to 0. lipid_sel (str,optional): atomselect string describing what should count

as “lipid”. Defaults to “lipid”
topologies (list of str): Topology files that were used in creating the
psf

prompt_topos (bool): Whether to ask for more topology files
